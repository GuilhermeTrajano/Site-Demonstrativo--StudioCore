# StudioCore — Site Institucional Demonstrativo

Site institucional **premium** de arquitetura e design de interiores, desenvolvido como
projeto demonstrativo pela [**GT Core**](https://gtcore.vercel.app/).

> **StudioCore** é uma empresa fictícia. O objetivo do projeto é demonstrar capacidade de
> desenvolvimento de sites institucionais modernos, com foco em **Landing Pages, SEO,
> Responsividade e Conversão**.

## ✨ Destaques

- **Design premium** inspirado em Apple, Framer e Awwwards — minimalista e sofisticado
- **100% responsivo** (desktop, tablet e mobile) com menu mobile
- **SEO técnico**: meta tags, Open Graph, Twitter Cards e dados estruturados (JSON-LD)
- **Animações on-scroll**, contadores animados e hero com efeito de zoom
- **Filtro de projetos** por categoria (Residencial / Comercial / Corporativo)
- **Formulário de captação de leads** com validação (simulação, sem backend)

## 🗂 Estrutura

## ▶️ Como rodar localmente

O site é totalmente estático (sem backend). Basta abrir o `index.html` no navegador, ou
servir localmente:

```bash
# Python
python -m http.server 8000   # http://localhost:8000

# Node.js
npx serve
```
